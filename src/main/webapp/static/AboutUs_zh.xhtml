Fun Fun Spell 是一個提供免費英語練習的網站。成立於 2009 年 1 月。
它是由兩個香港的志願者發展，包括教授英語的父親以及當軟件工程師的兒子。
由於是義務工作性質，我們會以有限的資源， 不斷改善和發展 Fun Fun Spell。

您的支持是我們不斷改善和發展的主要動機 !

到現在為止，Fun Fun Spell 提供:

  1. 生字練習 - 學習英文生字和聽寫能力
  2. 拼音練習 - 練習國際音標 (IPA)
  3. 我的默書 - 提供發聲默書，不論你輸入任何英文生字
