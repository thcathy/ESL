Dictation System Phase 3 Updates

We have completed phase 3 development on our dictation system.
Phase 3 updates included the following:

1. You can input some specific characters in vocabulary. We support most of the vocabulary chacacters from Yahoo Dictionary.
  a) ' ' space: [bus stop]
  b) '-': [e-mail]
  c) '.': [E.Q.]
  d) '/': [bus stop]

2. Some dictations are shown in the first page which should increase the usage of dictation system.
  a) The Most Attempted Dictations
  b) The New Dictations
  c) The Highest Rating Dictations
  d) Last Dictation Practiced

3. A new member welcome page display your statistics of vocabulary practice, phonics practice plus dictations.
