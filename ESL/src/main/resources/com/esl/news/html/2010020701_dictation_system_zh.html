創建您自己的默書 - 給學生 /家長 /教師的默書系統

由於我們的本業的工作十分繁重，Fun Fun Spell 已經三個月沒有功能更新。
幾經努力下，總於在 2010 年新的開始創建了一個新的默書系統 。
這不只是更新現有的默書功能。這是一個全新的系統，可以保存默書和監測默書進程。此外，你亦可以使用其他人創造的默書。

對於導師或家長，您可以建立一個默書再把路徑 (URL) 發送給學生。在 Fun Fun Spell 網頁上， 你可以知道有誰做你的默書，並且知道他們得到的成績。
所有功能都可以在"我的聽寫"頁面裡面的新菜單找到，快來試試吧！

通過第一階段，我們已經推出：

  1. 建立 /編輯默書
  2. 給予默書評級
  3. 顯示您的記錄
  4. 顯示誰有做過默書

我們正在為第二階段努力，其中包括功能:

  1. 統計功能，例如最高評分的默書
  2. 隨機默書
  3. 刪除默書
  4. 還有...

請填寫聯絡我們的表格或電子郵件告訴我們您想要的任何功能。
